Getting Started with Nuxt

Nuxt.js is a powerful framework built on Vue.js that simplifies the development of modern web applications. It provides server-side rendering, static site generation, and a structured approach to building Vue apps.

Why Use Nuxt?

  • Automatic Routing: File-based routing makes navigation easy.
  • SEO-Friendly: Supports server-side rendering (SSR) for better search engine indexing.
  • Static Site Generation (SSG): Build pre-rendered pages for improved performance.
  • Vue Ecosystem: Built on Vue.js, making it familiar to Vue developers.

Installing Nuxt

To create a new Nuxt project, you need to have Node.js installed. Run the following command to set up a new Nuxt app:

npx nuxi init my-nuxt-app
cd my-nuxt-app
npm install
npm run dev

This will start a local development server at http://localhost:3000.

Project Structure

A typical Nuxt project has the following structure:

my-nuxt-app/
├── pages/        # Vue files that define routes
├── components/   # Reusable UI components
├── layouts/      # Layouts for structuring pages
├── store/       # Vuex store for state management
├── plugins/      # External plugins
├── static/       # Static assets (e.g., images, fonts)
├── nuxt.config.ts # Configuration file

Creating a Page

Nuxt automatically generates routes based on the pages/ directory. Create a new page at pages/about.vue:

<template>
  <div>
    <h1>About Us</h1>
    <p>Welcome to our Nuxt application!</p>
  </div>
</template>

Now, visiting /about in the browser will show this page.

Fetching Data in Nuxt

Nuxt provides useFetch and useAsyncData for fetching data in pages. Here's an example:

<script setup>
const { data } = useFetch('https://jsonplaceholder.typicode.com/posts/1')
</script>

<template>
  <div>
    <h1>{{ data.title }}</h1>
    <p>{{ data.body }}</p>
  </div>
</template>

Deploying Your Nuxt App

To generate a static site, run:

npm run generate

This will create a dist/ folder with your fully static website, ready for deployment on platforms like Vercel, Netlify, or GitHub Pages.

Conclusion

Nuxt.js makes it easier to build fast, SEO-friendly, and scalable Vue applications. Whether you're building an SSR-based app, an SSG-powered website, or a hybrid solution, Nuxt provides the flexibility and power needed.
